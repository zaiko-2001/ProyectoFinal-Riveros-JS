Proyecto Final - Curso de JavaScript en Coderhouse
Este proyecto es la entrega final del curso de JavaScript en Coderhouse. Consiste en un simulador de carrito de compras que implementa el DOM, el manejo de eventos, AJAX y JSON.

Funcionalidades
Renderizado de Productos: Los productos se renderizan automáticamente al cargar la página mediante un fetch asincrono.
Carrito de Compras: El carrito permite visualizar los productos agregados y cuenta con una función para vaciarlo.
Interfaz de Usuario: La interfaz está diseñada con Bootstrap para un diseño responsivo y atractivo.
Tecnologías Utilizadas
HTML
CSS
JavaScript
Bootstrap
AJAX
JSON
